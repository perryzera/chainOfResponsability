# Projeto Chain of Responsibility (Java)

## Como Executar

**Pré-requisito:** Ter o **JDK** (Java Development Kit) instalado.

1.  Abra seu terminal na pasta do projeto.

2.  Compile todos os arquivos `.java`:
    ```bash
    javac *.java
    ```

3.  Execute a classe `TesteCadeia`:
    ```bash
    java TesteCadeia
    ```

## Saída Esperada
--- Testando compra de R$ 500 --- Compra autorizada para o comprador

--- Testando compra de R$ 5.000 --- Compra autorizada para o gerente

--- Testando compra de R$ 100.000 --- Compra autorizada para o diretor
